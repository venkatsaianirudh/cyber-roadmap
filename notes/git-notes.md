What is git?
git is used as a version control system.
it tracks the changes in your files, acts as checkpoints for your work.

	Git commands

	1. git innit
	## Makes the current folder a Git repository.

	2. git add .
	## Takes all changed files and prepare them for the next commit.

	3. git commit
	## creates a saved checkpoint.

	4. git push
	## updates the git repository from local commits.
	
	5. git log
	## Shows commit history.
	
	6. git status
	## Shows curreent reeepo state and changed files.
