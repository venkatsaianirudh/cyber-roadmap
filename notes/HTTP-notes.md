HTTP Intro

# Request = What your browser sends.
# Response = What the server sends back
# headers = Extra info attached to requests/responses.
# Methods = actions like GET or POST.
#Status codes = result of the request.

##IMPORTANT CODES AND METHODS.
* GET = ask for data/page
* POST = send data
* 200 = success
* 301/302 = redirect
* 403 = forbidden
* 404 = not found
* 500 = server error

## to see the methods, status codes and information on request and response headers, open a brouser inspect a website and navigate to the network tab.
