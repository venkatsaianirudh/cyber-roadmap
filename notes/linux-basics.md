### Linux starter commands. ###


		1.  pwd
		##Shows current folder path.

		2.  ls 
		## Shows whats inside the current folder. 
		## "ls -1" makes the list vertical.
		## "ls -a" shows hidden files. 
		## "ls -l" information about files and directories in the file system.

		3.  cd 
		## Changes directory.

		4.  mkdir 
		## Makes a Folder.

		5.  touch 
		## Makes a file.

		6.  cp
		## Copies a file. also need to add the designated file you would need to copy the origin file to.

		7.  mv 
		## Moves or renames a file.

		8.  rm
		## Deletes a file.
		
		9.  rmdir
		## removes an empty dirctory.
		## if the folder has files in it, use "rm -r foldername". "rm -rf foldername" for a force delete.
		
		10. cat
		## Shows file contents in terminal.

		11. less
		## Lets you scroll through a file.

		12. head
		## Shows first lines of a file.

		13. tail
		## Shows last lines of a file.

		14. chmod
		## Changes file permissions.
		## can make files executable, read, or write or levels of authorization.
		
		15. ps
		## Shows running processes.
		
		16. top 
		## Live process view (think of task manager.)
