#Understanding Burp suit

## What I did
* opened burp suite.
* navigated to proxy
* turned interceptor on.
* Opened Youtube.com
* Capturted a request.
* Inspected the elements of the captured request.
* forwarded the request.
* Turned the Interceptor off.

## What I observed
* The method of the request captured.
* URL
* Headers of the request seen
* status code after forwarding the request.

## What I learned
* Burp suite can capture browser requests before they got to the server.
* When the interceptor is turned on, the request is held in burpsuite until forwarded.
